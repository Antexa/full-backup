Scripts de sauvegarde/restauration système sous Linux
==========================================================

### Installation

```bash
apt-get update && apt-get dist-upgrade
apt-get install git-core
```

```bash
cd /opt
git clone https://github.com/hardware/full-backup.git
cd full-backup
chmod +x *.sh && ./install.sh
```
